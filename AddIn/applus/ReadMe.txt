Архив содержит 2 компоненты "APPlus.dll" и "APPlusCOM.dll".
1) APPlus.dll - предназначена для обеспечения механизмов взаимодействия  1С:Предприятия с ПО "Агент Плюс" версий 1.5 и 2.0 (в файл-серверном режиме). Компоненту необходимо скопировать в папку базы данных 1С:Предприятия для 1С 7.7 и в папку установки - для 1С 8. Для правильного функционирования компоненты в папке вместе компонентой должны находиться следующие файлы:
 1   mfc100.dll
 2.  mfc100u.dll
 3.  msvcp100.dll
 4.  msvcr100.dll
 5.  1CPP.dll

2) папка "APPlusCOM" - предназначена для обмена данными в фоновом режиме при использовании 1С:Предприятия 8 (в клиент-серверном режиме).
    2.1. При работе с 32-разрядным Сервером 1С:Предприятия необходимо использовать файл "APPlusCOM32.dll".
    2.2. При работе с 64-разрядным Сервером 1С:Предприятия необходимо использовать файлы "APPlusCOM32.dll" и "APPlusCOM64.dll".

Подробная информация о процедуре регистрации компонент приведена в документации к конфигурациям 1С.

3) APPlus.als - файл описания методов компоненты "APPLus.dll" для 1С:Предприятия 7.7. Файл необходимо скопировать в папку базы данных 1С:Предприятия. Просмотр содержимого файла осуществляется в Конфигураторе 1С в Синтакс-Помощнике.

4) APPlus.chm - описание методов компоненты "APPLus.dll".
